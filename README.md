# MATLAB Mobile Data Transmission Simulation in a 3D Urban Wireless Environment

## 1. Project Title

**Development of a MATLAB Application for Simulation and Performance Evaluation of Mobile Data Transmission in a 3D Urban Wireless Environment**

Tên tiếng Việt:

**Xây dựng ứng dụng MATLAB mô phỏng và đánh giá truyền dữ liệu di động trong môi trường vô tuyến đô thị 3D**

---

## 2. Project Overview

This project is a MATLAB-based simulation application for visualizing and evaluating the transmission of mobile digital data through a wireless communication system.

The application models the full communication chain from digital input bits to channel coding, digital modulation, RF signal generation, 3D urban wireless propagation, receiver front-end processing, digital demodulation, channel decoding, and final performance evaluation.

The main goal of this project is to help students and engineers understand how digital data is transformed into RF signals and how the wireless environment affects the received signal quality.

---
